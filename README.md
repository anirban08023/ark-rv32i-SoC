# RISC-V SoC Implementation and Simulation

This repository contains the project for implementing a full RISC-V SoC in Verilog and simulating it with Verilator.

## 🔍 Overview
This project is focused on implementing a full RISC-V SoC in Verilog for simulation in Verilator. The scope includes creating a vector processing module and other necessary functionalities to build a complete SoC model.

## 📂 Repository Structure
- **rtl**: Contains the Verilog source code for the RISC-V core and SoC components.
- **tb**: Testbenches written in C++ for simulating the design with Verilator.
- **sim**: Scripts and Makefiles for running simulations.
- **doc**: Supporting documentation and project resources.

## 🛠 Technologies Used
- **Verilog**: For the hardware description of the SoC.
- **C++**: For the Verilator testbench.
- **Verilator**: For compiling Verilog to a C++ model for simulation.
